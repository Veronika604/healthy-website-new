# Veronika_health — новый сайт

Готовый сайт со всеми страницами, новым дизайном и сохранением данных
пользователя в браузере (localStorage).

## Что внутри

```
new-site/
├── index.html          ← главная
├── register.html       ← регистрация (имя + email)
├── form.html           ← анкета (рост, вес, цели…)
├── menu.html           ← меню дня
├── profile.html        ← личный кабинет с дашбордом
├── css/
│   └── style.css       ← все стили
├── js/
│   └── app.js          ← вся логика
└── images/             ← сюда положить картинки
```

## Как загрузить на GitHub (новый репозиторий)

### Шаг 1. Создай новый репозиторий
1. Зайди на github.com и нажми зелёную кнопку **New** (или + → New repository).
2. Имя репозитория — любое, например `veronika-health-new` или `healthy-v2`.
3. Поставь галочку **Public** (обязательно для GitHub Pages).
4. Нажми **Create repository**.

### Шаг 2. Загрузи файлы
1. На странице нового репозитория нажми ссылку **uploading an existing file**
   (или Add file → Upload files).
2. Перетащи в окно **ВСЕ** файлы и папки из `new-site/`:
   - 5 HTML файлов (index, register, form, menu, profile)
   - папку `css` целиком
   - папку `js` целиком
   - папку `images` со всеми картинками
3. Прокрути вниз, нажми зелёную кнопку **Commit changes**.

### Шаг 3. Включи GitHub Pages
1. В репозитории нажми **Settings** (шестерёнка, справа сверху).
2. В левом меню выбери **Pages**.
3. В разделе **Source** выбери:
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажми **Save**.
5. Подожди минуту. Вверху появится зелёная надпись со ссылкой вида
   `https://твой-логин.github.io/имя-репозитория/`

Готово! Сайт в интернете.

## Как это работает (кратко)

1. **Регистрация** сохраняет имя и email в браузере пользователя.
2. **Анкета** сохраняет рост, вес, цели, диету, аллергии.
3. **Меню** фильтруется под анкету:
   - Веган → только веганские меню
   - Пескетарианец → + рыба
   - Аллергия на молоко → меню с йогуртом исключается
   - Калории корректируются: −15% для похудения, +20% для набора
4. **Личный кабинет** показывает:
   - Приветствие по имени
   - Вес, ИМТ (со шкалой), цель, калории на день
   - Меню дня
   - Тренировки с галочками (прогресс сохраняется)
   - Все ответы анкеты
   - Кнопку «Выйти» (удаляет все данные)

## Ограничения

Данные хранятся **только в браузере пользователя**. Это значит:
- На другом устройстве данных не будет
- Если очистить историю браузера — данные пропадут
- Ты не видишь список зарегистрированных

Для учебного / портфолио-проекта этого достаточно.
Если понадобится настоящая база данных — можно позже добавить Supabase.

## Как добавить новое меню

Открой `js/app.js`, найди массив `allMenus` в начале файла.
Добавь ещё один блок по образцу:

```javascript
{
  id: 4,
  diets: ['normal', 'keto'],
  allergens: ['dairy'],
  meals: [
    {meal:"Завтрак", name:"Омлет с беконом", kcal:420, protein:25, fat:32, carbs:4, img:"images/omelette.jpg"},
    {meal:"Обед",    name:"Салат Цезарь",    kcal:550, protein:40, fat:35, carbs:8, img:"images/caesar.jpg"},
    {meal:"Ужин",    name:"Стейк с овощами", kcal:600, protein:45, fat:40, carbs:10, img:"images/steak.jpg"}
  ]
}
```

Значения `diets`: `normal`, `vegetarian`, `vegan`, `pescetarian`, `glutenfree`, `keto`.
Значения `allergens`: `gluten`, `dairy`, `fish`, `soy`, `nuts`.
